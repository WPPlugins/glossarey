Glossary of your custom terms and terminology for wordpress

== Description ==

Glossarey is a custom dictionary plugin for your wordpress site. It's functionality is very basic - you create a list of your terms through admin Glossary post type, then when creating post you can use the shortcode generator button built in the WP Editor. Plugin will generate a link, which will display a tooltip on mousover action while hovering the link.

== Installation ==

1. Upload `glossarey/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
